# Setup API

## Environment Variables

-   Create a `.env` file in the `packages/infra` directory
-   Add these required variables and give them values
    -   CDK_DEFAULT_ACCOUNT
    -   CDK_DEFAULT_REGION
    -   DOMAIN_NAME
    -   APP_NAME
    -   STAGE

## AWS

-   Install the AWS CLI
-   Setup a Profile for your AWS account
-   Setup a public hosted zone in Route53
